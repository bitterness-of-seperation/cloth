# 启动脚本使用说明

本项目提供了三种启动脚本，可以一键启动前后端服务。

## 脚本列表

### 1. Windows 批处理脚本 (推荐 Windows 用户)
**文件**: `start-dev.bat`

**使用方法**:
```bash
# 双击运行，或在命令行中执行
start-dev.bat
```

**特点**:
- 自动检查 Python 和 Node.js 环境
- 在新窗口中启动前后端服务
- 关闭窗口即可停止服务

---

### 2. PowerShell 脚本
**文件**: `start-dev.ps1`

**使用方法**:
```powershell
# 在 PowerShell 中执行
.\start-dev.ps1
```

**注意**: 首次运行可能需要设置执行策略：
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

**特点**:
- 彩色输出，界面友好
- 自动检查环境
- 按 Ctrl+C 停止所有服务

---

### 3. Linux/Mac Shell 脚本
**文件**: `start-dev.sh`

**使用方法**:
```bash
# 添加执行权限
chmod +x start-dev.sh

# 运行脚本
./start-dev.sh
```

**特点**:
- 适用于 Linux 和 macOS
- 彩色输出
- 按 Ctrl+C 停止所有服务

---

## 服务地址

启动成功后，可以访问：

- **前端**: http://localhost:3000
- **后端**: http://localhost:8000
- **API 文档**: http://localhost:8000/docs

## 前置要求

### 必须安装:
1. **Python 3.8+**
   - Windows: https://www.python.org/downloads/
   - Linux: `sudo apt install python3 python3-pip python3-venv`
   - macOS: `brew install python3`

2. **Node.js 18+**
   - 官网: https://nodejs.org/
   - 或使用 nvm: `nvm install 18`

### 首次运行:
脚本会自动：
- 创建 Python 虚拟环境
- 安装后端依赖
- 安装前端依赖

---

## 手动启动（不使用脚本）

### 启动后端:
```bash
cd backend
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 启动前端:
```bash
cd frontend
npm install
npm run dev
```

## 常见问题

### Q: 端口被占用怎么办？
**A**: 修改端口号

后端端口 (8000):
- 修改启动命令中的 `--port 8000`

前端端口 (3000):
- Next.js 会自动使用下一个可用端口 (3001, 3002...)

### Q: Python 虚拟环境创建失败？
**A**: 确保安装了 `python3-venv`
```bash
# Ubuntu/Debian
sudo apt install python3-venv

# 或使用系统 Python
python -m pip install --user virtualenv
```

### Q: npm install 很慢？
**A**: 使用国内镜像
```bash
npm config set registry https://registry.npmmirror.com
```

### Q: 后端启动失败？
**A**: 检查依赖安装
```bash
cd backend
source .venv/bin/activate
pip install -r requirements.txt
```

### Q: 前端启动失败？
**A**: 清除缓存重试
```bash
cd frontend
rm -rf node_modules .next
npm install
npm run dev
```

## 停止服务

### 使用脚本启动:
- Windows 批处理: 关闭命令行窗口
- PowerShell/Shell: 按 `Ctrl+C`

### 手动启动:
在各自的终端窗口按 `Ctrl+C`
