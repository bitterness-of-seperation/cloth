# 虚拟试穿 App - 项目目标文档

## 项目概述

一个基于 Web 的虚拟试穿应用，让用户可以通过上传照片或实时 AR 方式预览服装穿着效果，并在社区中分享和讨论。

## 核心功能模块

### 1. 登录注册

- 邮箱/密码注册登录
- 第三方登录 (Google / GitHub)
- 用户资料管理 (头像、昵称、简介)
- 路由守卫 (未登录重定向)

### 2. 图片试穿 (AI 合成)

- 上传人物全身照
- 选择或上传服装图片
- 调用 AI 模型合成试穿效果图
- 查看、保存、分享合成结果

### 3. 实时 AR 试穿

- 摄像头实时捕获画面
- MediaPipe 人体姿态关键点检测
- 服装图片根据关键点实时叠加
- 拍照保存 AR 试穿效果

### 4. 社区分享讨论

- 发帖 (文字 + 图片，可关联试穿结果)
- 帖子列表 (瀑布流展示)
- 评论
- 点赞
- 个人主页 (我的帖子、试穿记录)

## 技术方案

### 前端

- **框架**: Next.js 14 (App Router) + TypeScript
- **样式**: Tailwind CSS + shadcn/ui
- **状态管理**: Zustand
- **AR**: MediaPipe Pose + Canvas

### 后端

- **BaaS**: Supabase (Auth + PostgreSQL + Storage)
- **AI 服务**: Python FastAPI
- **AI 模型**: IDM-VTON / OOTDiffusion (虚拟试穿模型)

### 数据库表

| 表名 | 说明 | 关键字段 |
|------|------|----------|
| profiles | 用户资料 | id, username, avatar_url, bio |
| clothing_items | 服装库 | id, name, image_url, category, user_id |
| tryon_results | 试穿结果 | id, user_id, person_image_url, clothing_id, result_image_url |
| posts | 帖子 | id, user_id, content, image_urls, tryon_result_id |
| comments | 评论 | id, post_id, user_id, content |
| likes | 点赞 | id, post_id, user_id |

### Storage Buckets

- `avatars` - 用户头像
- `clothing` - 服装图片
- `person-photos` - 用户上传的人物照片
- `tryon-results` - AI 合成的试穿结果
- `post-images` - 帖子图片

## 部署方案

- **前端**: Vercel
- **AI 后端**: 云服务器 (GPU)
- **数据库/认证/存储**: Supabase Cloud
